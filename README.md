# 闲鱼选品上品助手 MVP

这是一个无依赖 Chrome Extension 原型。

它优先解决最核心的 7 件事：

1. 识别闲鱼 / 1688 / 拼多多 / 淘宝商品页
2. 采集商品基础信息
3. 计算成本、利润、利润率
4. 产出规则化综合评分
5. 生成闲鱼风格标题、描述、标签和价格建议
6. 保存商品库
7. 保存草稿箱并在后台管理

## 当前实现范围

### 插件端

- `当前商品`
  - 自动识别当前页面商品详情页的标题、价格、图片、发货地、销量线索
  - 显示平台、状态、疑似重复提示
  - 支持一键采集

- `选品分析`
  - 输入货源成本、运费、包装成本、预计售价
  - 调整需求、竞争、风险、履约等级
  - 输出预计利润、利润率、综合评分、风险提示

- `上品内容`
  - 生成 5 个标题版本
  - 生成闲鱼发布描述
  - 给出标签建议和价格建议
  - 支持复制标题、复制描述、复制全部、保存草稿

- `商品库`
  - 查看最近采集商品
  - 平台 / 状态筛选
  - 载入已采集商品继续编辑

### 后台页

- 首页看板
  - 今日采集数
  - 高分商品数
  - 待上架商品数
  - 已上架商品数
  - 最近 7 天趋势
  - 类目分布

- 商品库
  - 搜索、平台筛选、状态筛选
  - 标记已上架 / 已淘汰
  - 重新生成内容
  - 一键生成或更新草稿

- 草稿箱
  - 查看草稿标题、价格、标签、版本号
  - 修改描述
  - 复制整份草稿
  - 标记已完成

## 如何加载

1. 打开 Chrome，进入 `chrome://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录 `下载地址\FishHelper`
5. 打开一个闲鱼 / 1688 / 拼多多 / 淘宝商品详情页(目前只测试了咸鱼)
6. 点击扩展图标，开始测试

## 数据存储方式

当前版本所有数据保存在 `chrome.storage.local`：

- 商品主表：`products`
- 草稿表：`drafts`
- 界面偏好：`preferences`
